# 项目简介：
本项目为基于 C++ 语言与 Qt 跨平台图形界面框架实现的经典策略棋盘游戏《五子棋》。在开发过程中，我们综合运用了面向对象程序设计（OOP）思想、Qt 核心绘图系统（QPainter）、事件驱动机制（鼠标事件与信号槽）等多方面知识，配合 CLion 集成开发环境与 CMake 构建工具实现模块化工程管理，构建出一个逻辑完整、交互流畅的桌面端五子棋游戏。
通过本项目的开发，我们不仅加深了对 C++ 语言核心语法与类结构设计的理解，还掌握了 Qt 框架在实际 GUI 项目中的核心应用方式，例如通过重写 paintEvent 实现棋盘与棋子的 2D 绘制、利用算法实现多方向的胜负判定以及人机对战中的决策逻辑。同时，团队成员也在协作开发、代码调试、CMake 配置以及本地版本管理等方面获得了宝贵的实战经验，增强了对事件循环机制和现代图形界面开发流程的整体把握。整个项目过程鼓励我们探索课外内容、查阅 Qt 官方文档、分析优秀开源代码，从中学习他人的界面设计思路与算法实现方法，从而有效提升了编程能力和工程实践水平。

##主要功能：
（1）账号注册，登录和找回

设置账号密码，和找回密码的问题与答案，选择游客登录。

（2）双人对战

实现对局时长的设置，棋盘绘制，悔棋，求和，暂停，保存记录功能

（3）人机对战

实现悔棋，人机对战，保存记录功能

（4）双人对战记录查看

查看已保存的双人对局记录

（5）人机对战记录查看

查看已保存的人机对局记录
